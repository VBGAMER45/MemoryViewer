1. Introduction
Memory viewer and dumper allows you to view an applications memory
and allows you to the dump the memory to a file. You always get free updates
to the next version and you are welcome to ask for a feature to be added.

2. Features
-Allows to view memory of a running process.
-Allows you to save the memory dump to a file.
